# 🕰️ 隱形專注小助手 (Invisible Focus Timer)

這是一個專為需要高度專注的專業人士設計的 Python 自動化腳本。它能幫助你執行「工作 30 分鐘、休息 10/20 分鐘」的循環，並且在背景完全隱身，不佔用任何桌面空間。

## ✨ 特色功能
- **完全隱身**：執行後不會彈出黑色視窗或程式介面。
- **智能循環**：
  - 每工作 30 分鐘提醒一次。
  - 前兩次休息為 10 分鐘（短休息）。
  - 第三次休息為 20 分鐘（長休息）。
- **優雅提醒**：使用 Windows 內建「星際感」音效，搭配系統彈窗提醒。
- **防干擾機制**：若你在忙碌（如全螢幕簡報），提醒會自動延後直到你空閒。

## 🚀 如何使用
1. **安裝環境**：確保電腦已安裝 [Python](https://www.python.org/)。
2. **下載檔案**：下載 `my_timer.pyw` 到你的桌面。
3. **啟動程式**：連點兩下 `my_timer.pyw` 即可。
4. **確認執行**：打開「工作管理員」>「詳細資料」，看到 `pythonw.exe` 代表運作中。

## 🛑 如何關閉
由於程式採隱身設計，若要結束請執行以下步驟：
1. 按下 `Ctrl + Shift + Esc` 開啟 **工作管理員**。
2. 切換到 **詳細資料** 分頁。
3. 找到 `pythonw.exe`，按右鍵選擇 **結束工作**。

## ⚙️ 自定義設定
你可以用記事本打開檔案，修改最上方的參數：
- `WORK_MINUTES`: 修改工作時間（預設 30）。
- `SHORT_BREAK_MINUTES`: 修改短休息時間（預設 10）。
- `LONG_BREAK_MINUTES`: 修改長休息時間（預設 20）。
